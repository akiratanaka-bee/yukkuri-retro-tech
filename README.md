# 深掘り！ロストテクノロジー図鑑（仮） プロジェクト設計書

## 1. チャンネルコンセプト
**「不便こそ、最高の技術革新だった。」**

現代の「なんでもできるデジタル技術」の視点から、かつての「制約だらけのアナログ技術」を再評価する。
単なる「懐古趣味」ではなく、**「なぜ当時の技術者はその設計にしたのか？」「メモリ数KBの中でどうやって動かしていたのか？」**という技術的創意工夫（ハック）に焦点を当て、先人の知恵へのリスペクトを提供する。

- **キーワード:** 制約の美学、物理ギミック、エンジニア魂、リスペクト
- **読後感:** 「昔の人は不便で可哀想」ではなく「この環境でこれを実現した昔の人は天才だ」という感動。

## 3. 動画フォーマット（ゆっくり解説）

### キャラクター設定
* **霊夢（現代視点）:** 素朴な疑問、ツッコミ役。「えっ、これ手動なの？不便すぎない？」
* **魔理沙（技術視点）:** 解説、熱弁役。「甘いぜ霊夢。この不便さの中に技術者の執念があるんだ」

### 構成テンプレート
1. **導入:** 現代との対比で興味を惹く
2. **問題提起:** 当時の絶望的な制約（メモリ不足、GPSなし等）
3. **解決策(Core):** 物理的・論理的な工夫の図解解説
4. **考察:** 現代技術への繋がり
5. **結び:** 技術へのリスペクト

## 4. コンテンツカテゴリ案

### A. 【物理メディアの謎】
* VHS vs ベータ：画質を捨てて掴んだ「録画時間」という戦略
* カセットテープ：物理的反転が生んだアルバム構成の美学

### B. 【通信・インフラの裏側】
* 黒電話：停電でも使える最強の給電システム
* 自動改札機：0.7秒で切符を整列させる磁気と重力の魔術

### C. 【ガジェット・ハードウェア】
* ウォークマン：機能を「削ぎ落とす」イノベーション
* ゲームボーイ通信ケーブル：物理接続が生んだ文化

### D. 【ロストテクノロジー・失敗】
* コンコルド：速すぎた技術と経済の壁
* プラズマテレビ：自発光の理想と敗北

## 5. デザイン方針
* **サムネイル:** レトロな実機写真 × 黒・金・赤の重厚感ある配色。「実はこれが世界を変えました」等の煽り。
* **図解:** 内部構造や動作原理のアニメーションを重視。
